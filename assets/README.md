# DuilioCode Studio Assets

## Logo

The DuilioCode Studio logo features **Duilio**, a friendly gnome wizard coding on multiple screens.

### Logo Files

- `logo.png` - Full logo (512x512)
- `logo-small.png` - Small version (128x128)
- `favicon.ico` - Browser favicon

### Usage Guidelines

1. **Do** use the logo with adequate spacing
2. **Do** maintain aspect ratio when resizing
3. **Don't** modify or distort the logo
4. **Don't** use low-quality versions

### Brand Colors

- Primary Blue: `#1E90FF`
- Accent Cyan: `#39c5cf`
- Background Dark: `#0d1117`

### Note

When using the provided logo image, please ensure to:
- Crop any watermarks from the bottom-right corner
- Use the central portion focusing on the Duilio gnome character
- Maintain the circular badge style
