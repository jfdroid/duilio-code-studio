# Introduction to DuilioCode Studio

## What is DuilioCode Studio?

DuilioCode Studio is an AI-powered code assistant that runs **locally on your computer**. It understands what you want to do and can **create, modify, delete, and read files** directly, without you needing to do it manually.

## Main Components

### 1. **Ollama** - AI Engine
- Local server that runs language models
- Communicates via HTTP API
- Supports various models (Qwen, Llama, etc.)

### 2. **Qwen2.5-Coder** - AI Model
- Code-specialized model
- Understands Python, JavaScript, TypeScript, Kotlin, etc.
- Generates functional and correct code

### 3. **FastAPI** - Web Framework
- Creates the API that connects frontend and backend
- Manages HTTP requests
- Processes and returns responses

### 4. **Action Processor** - Action Executor
- Reads actions generated by AI
- Executes on the file system
- Validates and protects against dangerous operations
